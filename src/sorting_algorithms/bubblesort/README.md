# Bubble Sort

Implementação do algoritmo **Bubble Sort** em Java.

## Sobre

O Bubble Sort é um algoritmo de ordenação simples que percorre o array várias vezes, comparando elementos adjacentes e trocando-os de lugar caso estejam na ordem incorreta. Esse processo é repetido até que nenhum elemento precise mais ser trocado.

## Como funciona

1. Comparar elementos vizinhos no array.
2. Trocar de posição se o elemento atual for maior que o próximo.
3. Repetir o processo para toda a lista até que ela esteja totalmente ordenada.
