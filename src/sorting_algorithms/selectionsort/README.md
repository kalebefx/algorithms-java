# Selection Sort

Implementação do algoritmo **Selection Sort** em Java.

## Sobre

O Selection Sort é um algoritmo de ordenação que percorre o array procurando o menor elemento e o posiciona no início da lista. Esse processo é repetido até que todos os elementos estejam ordenados.

## Como funciona

1. Encontrar o menor elemento do array.
2. Trocar com a posição atual.
3. Repetir o processo para o restante da lista.
